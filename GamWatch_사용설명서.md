# GamWatch 사용설명서

## GamWatch란?

국정감사 기간 중 게임 산업 관련 발언을 **자동으로** 모니터링하는 시스템입니다.

국정감사 기간 중 매일 새벽 2시에 자동으로 국회방송 유튜브 영상을 검색하고, 자막을 추출하여 AI가 게임 관련 안건을 분류·요약합니다. 결과는 대시보드에서 바로 확인하고, Slack 보고서로도 생성할 수 있습니다.

**핵심 흐름:**
유튜브 영상 → 자막 자동 추출 → AI 분석 → 대시보드 표시

---

## 1. 접속 방법

https://kipeum86.github.io/gamwatch-national-audit-monitoring/

처음 접속하면 **비밀번호 입력 화면**이 나타납니다.
전달받은 비밀번호를 입력하고 **확인**을 누르면 대시보드가 열립니다.

- 비밀번호는 브라우저에 저장됩니다 (같은 브라우저에서는 다시 입력 불필요)
- 다른 컴퓨터/브라우저에서는 다시 입력해야 합니다

---

## 2. 최초 1회 설정 (PAT 입력)

파이프라인 실행 기능을 사용하려면 처음 한 번만 설정이 필요합니다.

1. 대시보드 우측 상단의 **⚙ (톱니바퀴) 버튼** 클릭
2. 전달받은 **PAT 코드**를 입력
3. **저장** 클릭

PAT도 브라우저에 저장됩니다. 다른 컴퓨터/브라우저에서는 다시 입력해야 합니다.

---

## 3. 자동 모니터링 (핵심 기능)

**국정감사 기간 중에는 별도 조작 없이 매일 자동 실행됩니다.**

- 매일 새벽 2시(KST)에 자동으로 실행
- 대상 상임위(정무위, 과방위, 문체위, 산자위 등)의 국감 영상을 검색
- 새 영상이 있으면 자막 추출 → AI 분석 → 대시보드 반영

> **참고:** 자동 실행은 국정감사 기간(설정 파일에 지정)에만 동작합니다. 기간 외에는 자동 실행이 스킵되며, 수동 영상 분석 요청과 전체 파이프라인 실행 버튼은 기간과 관계없이 항상 사용할 수 있습니다.

국감 당일에 영상이 빨리 올라온 경우, 새벽까지 기다리지 않고 **"전체 파이프라인 실행"** 버튼을 눌러 즉시 처리할 수도 있습니다.

---

## 4. 대시보드 — 결과 확인

대시보드에 접속하면 AI가 정리한 국정감사 안건 목록을 확인할 수 있습니다.

### 카드 읽는 법

- **파란색 좌측 테두리** → 게임 관련 안건
- **우측 상단 ★ 표시** → 주요 게임사(넥슨, 엔씨소프트, 넷마블 등) 이름이 언급된 안건
- 카드를 클릭하면 상세 내용이 펼쳐집니다
  - 질의 의원 이름, 소속 정당, 발언 요약
  - 답변자(장관, 기관장 등)의 응답 요약
  - 관련 뉴스 기사 링크

### 필터 사용법

상단 필터바에서 원하는 조건으로 골라볼 수 있습니다.

| 필터 | 설명 |
|------|------|
| 날짜 | 특정 날짜의 안건만 보기 |
| 상임위 | 특정 상임위(문체위, 과방위 등)만 보기 |
| 분류 | "게임 관련"만 또는 "일반"만 보기 |
| 게임 관련만 | 버튼 클릭으로 게임 안건만 빠르게 필터링 |
| 검색 | 키워드로 안건 검색 (의원 이름, 회사명, 정책 키워드 등) |
| 초기화 | 모든 필터를 원래 상태로 되돌림 |

### 통계 요약

필터바 아래에 전체 건수, 게임 관련 건수, 게임사 언급 건수가 표시됩니다.

## 5. 수동 영상 분석 (필요 시)

자동 모니터링 대상이 아닌 영상(간담회, 업무보고 등)을 직접 분석할 수 있습니다.

1. 상단의 **"영상 분석 요청"** 버튼 클릭
2. 팝업에서 정보 입력
   - **유튜브 URL** (필수): 분석할 영상의 유튜브 링크
   - **이벤트 유형** (선택): 국정감사(기본), 업무보고, 간담회, 청문회 등 자유 입력 가능
   - **행사 일자** (선택): 기본값은 오늘 날짜
   - **상임위/기관명** (선택): 기존 상임위 자동완성 제공, 직접 입력도 가능 (예: 문체부, 청와대 등)
3. **"분석 요청"** 클릭

**자막은 서버에서 자동으로 추출합니다.** URL만 넣으면 됩니다.

자동 추출이 실패할 경우에만, "자막 직접 붙여넣기 (선택사항)" 영역을 펼쳐서 YouTube에서 복사한 자막을 직접 넣을 수 있습니다.

**자막 수동 붙여넣기 방법 (자동 추출 실패 시에만):**
1. YouTube 영상 페이지에서 영상 아래 **"...더보기"** 클릭
2. 펼쳐진 영역 하단의 **"스크립트 보기"** 클릭
3. 스크립트 패널에서 **Ctrl+A** (Mac: Cmd+A) → **Ctrl+C** 복사
4. 팝업의 "자막 직접 붙여넣기" 영역에 **Ctrl+V** 붙여넣기
5. **"분석 요청"** 클릭

### 진행 상황 확인

요청 후 상단에 파란색 상태 바가 나타납니다.

| 상태 | 의미 |
|------|------|
| 파이프라인 대기 중... | 서버에서 처리 준비 중 |
| 파이프라인 실행 중... | 자막 추출 및 AI 분석 진행 중 |
| 파이프라인 완료! | 분석 완료. 새로고침하면 결과 확인 가능 |
| 파이프라인 실패 | 오류 발생. "Actions에서 보기" 링크를 팀에 공유해 주세요 |

처리 시간은 영상 길이에 따라 보통 **10~30분**입니다.

## 6. 보고서 생성

대시보드에서 바로 Slack/메신저용 보고서를 생성할 수 있습니다.

1. 필터바의 **"보고서 생성"** 버튼 클릭
2. **날짜**와 **분류** 선택 → 해당 안건 목록이 체크박스로 표시됨
3. **보고서에 포함할 안건을 체크** (기본: 전체 선택, 불필요한 안건은 체크 해제)
4. **"선택한 안건으로 보고서 생성"** 클릭 → 미리보기 확인
5. **"클립보드 복사"** 클릭 → Slack이나 메신저에 붙여넣기

보고서는 상임위별로 안건과 주요 발언을 정리한 Slack 브리핑 양식으로 생성됩니다.
게임 관련 안건이 없는 날에도 분류를 "전체 안건"으로 선택하면 보고서를 만들 수 있습니다.

## 7. Google Sheets 사용 가이드

AI가 분석한 데이터는 Google Sheets에 저장됩니다. 오류 보정, 키워드 관리 등을 직접 할 수 있습니다.

> **주의: 각 탭의 1행(헤더)은 시스템이 사용하므로 절대 수정하지 마세요.** 헤더가 변경되면 시스템이 정상 동작하지 않습니다. (헤더 행은 편집 보호되어 있습니다.)

### agendas 탭 — 안건 목록

| 열 | 의미 | 편집 | 입력값 예시 |
|---|---|---|---|
| agenda_id | 안건 고유 ID | X (자동) | 2025-10-29-munche-01 |
| video_id | 영상 ID | X (자동) | BXdWgCwfUQs |
| committee | 상임위 이름 | X (자동) | 문화체육관광위원회 |
| date | 국감 일자 | X (자동) | 2025-10-29 |
| **category** | **분류** | **O** | **game** 또는 **general** |
| **title** | **안건 제목** | **O** | 게임산업 성장 및 세제 지원 |
| **summary** | **요약** | **O** | 게임 세제 지원 현황과 인재 육성 질의 |
| **is_company_mentioned** | **게임사 언급 여부** | **O** | **TRUE** 또는 **FALSE** |
| **company_mention_detail** | **언급 상세** | **O** | 넥슨, 크래프톤 세금 감면 관련 |
| sort_order | 표시 순서 | X (자동) | 1 |
| event_type | 이벤트 유형 | X (자동) | 국정감사 / 업무보고 / 간담회 등 |

**활용 예시:**
- AI가 "일반"으로 분류했지만 실제 게임 관련인 안건 → `category`를 `general`에서 `game`으로 변경
- 요약이 부정확한 경우 → `summary` 직접 수정

### statements 탭 — 발언 상세

| 열 | 의미 | 편집 | 입력값 예시 |
|---|---|---|---|
| statement_id | 발언 고유 ID | X (자동) | 2025-10-29-munche-01-s01 |
| agenda_id | 소속 안건 ID | X (자동) | 2025-10-29-munche-01 |
| **speaker_name** | **발언자 이름** | **O** | 홍길동 |
| **speaker_party** | **당적 약칭** | **O** | 민 / 국 / 조 / 진 / 개 / 무 (답변자는 빈칸) |
| **speaker_role** | **역할** | **O** | **questioner** (질의) 또는 **respondent** (답변) |
| **content** | **발언 내용 요약** | **O** | 게임산업 세제 지원 현황 질의 |
| sort_order | 표시 순서 | X (자동) | 1 |

**활용 예시:**
- 의원 이름이 잘못된 경우 → `speaker_name` 수정
- 당적이 누락된 경우 → `speaker_party`에 약칭 입력

### news_articles 탭 — 관련 뉴스

| 열 | 의미 | 편집 | 입력값 예시 |
|---|---|---|---|
| article_id | 기사 고유 ID | X (자동) | 2025-10-29-munche-01-n01 |
| agenda_id | 소속 안건 ID | X (자동) | 2025-10-29-munche-01 |
| **title** | **기사 제목** | **O** | 국감서 게임산업 세제 지원 논의 |
| **url** | **기사 URL** | **O** | https://news.example.com/... |
| **publisher** | **언론사** | **O** | 한국경제 |
| **published_at** | **발행일** | **O** | 2025-10-29 |

**활용 예시:**
- 관련 기사를 직접 추가하고 싶을 때 → 새 행 추가 후 `agenda_id`와 나머지 열 입력
- 무관한 기사 삭제 → 해당 행 삭제

### _keywords 탭 — 키워드 관리

| 열 | 의미 | 편집 | 입력값 예시 |
|---|---|---|---|
| **keyword** | **키워드** | **O** | 확률형 아이템 |
| **type** | **유형** | **O** | **include** 또는 **exclude** |
| **note** | **메모** | **O** | (선택) 설명 메모 |

- **include**: 이 키워드가 포함된 내용을 게임 관련으로 감지
- **exclude**: 이 키워드가 포함된 내용을 게임 관련에서 제외 (오탐 방지)

**예시:**

| keyword | type | note |
|---------|------|------|
| 확률형 아이템 | include | |
| 김OO 의원 | include | 게임 관련 발언이 잦은 의원 |
| 게임 체인저 | exclude | 게임과 무관한 관용 표현 |

키워드 변경 후 다음 파이프라인 실행 시 반영됩니다.

### _processed_videos 탭 — 처리 기록 (참고용)

시스템이 자동으로 기록합니다. **확인만 가능하고 수정은 불필요합니다.**

| 열 | 의미 |
|---|---|
| video_id | 영상 ID |
| status | 처리 결과 (success / no_subtitle / error) |
| subtitle_source | 자막 출처 (browser_extracted / notebooklm / auto_generated / stenographer) |
| processed_at | 처리 시각 |

## 자주 묻는 질문

**Q: 접속했는데 비밀번호를 모르겠어요.**
A: 팀 내 전달받은 비밀번호를 입력해 주세요. 모르시면 개발팀에 문의하세요.

**Q: 아무것도 안 했는데 데이터가 쌓여 있어요.**
A: 정상입니다. 매일 새벽 2시에 자동으로 실행되어 새로운 국감 영상을 분석합니다.

**Q: 데이터가 안 보여요.**
A: 아직 파이프라인이 실행되지 않았을 수 있습니다. "전체 파이프라인 실행" 버튼을 눌러주세요.

**Q: 영상 분석을 요청했는데 결과가 안 나와요.**
A: 상태 바를 확인해 주세요. "실행 중"이면 아직 처리 중입니다 (최대 30분). "실패"로 표시되면 "Actions에서 보기" 링크를 개발팀에 공유해 주세요.

**Q: ★ 표시는 뭔가요?**
A: 주요 게임사(넥슨, 엔씨소프트, 넷마블, 크래프톤 등) 이름이 직접 언급된 안건입니다.

**Q: 게임 관련인데 "일반"으로 분류된 안건이 있어요.**
A: AI 분류가 완벽하지 않을 수 있습니다. 두 가지 방법으로 보정할 수 있습니다:
- **즉시 보정**: Google Sheets의 agendas 탭에서 `category`를 `game`으로 수정 → 새로고침
- **향후 반영**: `_keywords` 탭에 관련 키워드를 추가

**Q: 게임 관련 안건이 없는 날에 보고서를 만들고 싶어요.**
A: 보고서 생성 팝업에서 분류를 "전체 안건"으로 선택하면 됩니다.

**Q: 다른 컴퓨터에서 접속했더니 파이프라인 실행이 안 돼요.**
A: ⚙ 설정 버튼에서 PAT 코드를 다시 입력해 주세요. PAT과 비밀번호는 브라우저별로 저장됩니다.

**Q: 보고서 복사가 안 돼요.**
A: "클립보드 복사" 버튼이 동작하지 않으면, 보고서 텍스트를 마우스로 직접 선택한 후 Ctrl+C로 복사해 주세요.

**Q: Sheets에서 헤더(1행)을 실수로 바꿨어요.**
A: 헤더 행은 보호되어 있어 일반적으로 수정이 안 됩니다. 만약 변경된 경우 개발팀에 문의하세요.
